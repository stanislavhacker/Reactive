What is Reactive?

Reactive is simple and fast library for working with html dom structure. You do not work directly with the dom, but using shadow dom, which is able to update the real dom based on performance and load the browser. This library does not use any framework or additional libraries and is very minimalist. An interesting feature is also generating css based dom.
